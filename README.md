# 코드카타(CodeKata)

A kata is an exercise in karate where you repeat a form many, many times, making little improvements in each.

카타는 하나의 양식을 여러 번 반복하면서 조금씩 개선하는 가라데의 운동입니다.  
SQL과 알고리즘 문제를 반복적으로 풀며 코딩 실력을 단련하고자 합니다.
